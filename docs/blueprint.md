# **App Name**: ElderLink

## Core Features:

- User Authentication: Secure phone/email login with voice-guided onboarding for seniors and profile linking.
- AI Photo Restoration: Restore old photos using generative AI. Serves as a memory tool and promotes engagement.
- Community Forum: Enable interest-based and location-based groups for seniors to connect and share experiences.
- SOS Emergency Response: One-tap emergency alerts with escalation to family and service providers. Supports immediate assistance.
- Telemedicine Integration: Book appointments, manage e-prescriptions and reminders all in one convenient place.
- Mood tracking with NLP: Use sentiment analysis as a tool for guardians, providing notifications of significant negative shifts in a senior's emotional state.
- Skills Marketplace: Connect seniors looking to share their skills in consulting or tutoring roles, empowering them and enriching the community.

## Style Guidelines:

- Primary color: Soft, calming blue (#64B5F6) to inspire trust and security.
- Background color: Light gray (#F5F5F5) for a clean and accessible interface.
- Accent color: Warm orange (#FFB74D) for important action items and highlights.
- Headline font: 'Alegreya', serif, for an elegant, intellectual, contemporary feel
- Body font: 'Inter', sans-serif, to ensure readability and simplicity.
- Use clear and universally recognizable icons, with sufficient contrast to improve legibility for elderly users.
- Maintain a clean, spacious layout to minimize cognitive load. Use a grid-based structure for consistent alignment and easy navigation.
